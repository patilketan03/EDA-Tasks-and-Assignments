# Analysis on ML Test Scores

## Project Objective
The objective of this project is to analyze Machine Learning test scores using Python and apply basic data analysis and visualization techniques to extract meaningful insights.

## Project Description
In this project, ML test score data is explored to understand score distribution and performance trends.  
The project demonstrates practical skills in data handling, exploratory data analysis (EDA), and visualization, which are essential for data-related internship roles.

## Skills & Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Jupyter Notebook  

## Key Tasks Performed
- Loaded and inspected the dataset  
- Cleaned and prepared the data  
- Performed Exploratory Data Analysis (EDA)  
- Created visualizations to analyze score patterns  

## Project Files
- `Task 1 - Analysis on ML Test Scores.ipynb` – Main analysis notebook  
- `README.md` – Project documentation  

## Learning Outcome
- Improved understanding of data analysis workflow  
- Hands-on experience with Python libraries  
- Ability to interpret data using visual insights
